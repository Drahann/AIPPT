# Component and Layout Reference

This document was normalized to English-only text to avoid encoding issues.

## Scope
- Theme packs and design tokens
- Figma-native cover/section/quote/ending templates
- Semantic slide layout components
- Atomic UI building blocks
- Layout-to-item-count routing rules

## Current Routing Rules
- `text-bullets`: 6-8 points
- `cards-split`: 3-5 items
- `list-featured`: 3-8 items
- `timeline`: 3-5 events
- `milestone-list`: 3-6 events
- `metrics`: 4-6 metrics
- `metrics-rings`: 1-3 metrics
- `chart-bar`: recommended 4-8 categories
- `chart-bar-compare`: exactly 2 series, recommended 4-8 categories
- `features-list-image`: recommended 3-4 items
- `quote`: 1-5 quotes
- `quote-no-avatar`: 1-5 quotes

## Notes
- Keep all source files in UTF-8.
- Avoid mixed encodings when copying text between editors or terminals.
- If Chinese copy is required later, add it explicitly with UTF-8 validation.
